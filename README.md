#  Docker Microservices Example

Projeto simples demonstrando como criar e executar containers utilizando Docker e Docker Compose.

---

##  Tecnologias utilizadas

* Docker
* Docker Compose
* Nginx

---

##  Estrutura do projeto

docker-microservices
│
├─ Dockerfile
├─ docker-compose.yml
├─ index.html
└─ README.md

---

##  Como executar

1. Clone o repositório:

git clone https://github.com/RyanHenrique3226/docker-microservices

2. Entre na pasta do projeto:

cd docker-microservices

3. Execute o container:

docker compose up --build

---

##  Acessar aplicação

Depois de iniciar o container, abra no navegador:

http://localhost:8080

---

##  Objetivo do projeto

Este projeto foi criado para demonstrar conceitos básicos de containerização utilizando Docker:

* criação de uma imagem com Dockerfile
* execução de containers
* utilização do Docker Compose
* estrutura básica de um projeto containerizado
